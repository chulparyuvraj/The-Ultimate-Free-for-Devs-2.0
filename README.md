# Curated Free Resources for Developers (2026 Edition)

## 1. Free AI & LLM APIs
*   **Google AI Studio (Gemini)**: Currently offering a generous free tier for Gemini 1.5 Flash and Pro (with rate limits). Best for large context windows.
*   **Groq**: Extremely fast inference for open-source models like Llama 3 and Mixtral. Free during their beta/growth phase.
*   **Hugging Face Inference API**: Free access to thousands of open-source models for NLP, vision, and audio tasks.
*   **OpenRouter**: Provides a unified API for various LLMs. While mostly paid, they often have free models available (e.g., Mistral, Phi-3).
*   **SiliconFlow**: Rising provider for high-performance AI model inference with a solid free tier for developers.

## 2. Free Hosting & Deployment (Frontend & Fullstack)
*   **Vercel**: The gold standard for Next.js and static sites. Includes serverless functions and edge middleware.
*   **Netlify**: Excellent for static sites and serverless functions. Offers great DX and edge features.
*   **Cloudflare Pages**: Unlimited bandwidth and seats for static sites. Highly recommended for production-grade free hosting.
*   **Render**: Good free tier for static sites, web services (Node.js, Python, Go), and PostgreSQL databases.
*   **Railway**: Offers a trial credit-based free tier that is very developer-friendly for deploying Docker containers.
*   **Zeabur**: Emerging platform for deploying full-stack apps with a focus on ease of use and a functional free tier.

## 3. Free Databases (SQL & NoSQL)
*   **Supabase**: The open-source Firebase alternative. Free tier includes PostgreSQL, Auth, Realtime, and Storage.
*   **Neon**: Serverless PostgreSQL with a generous free tier, including database branching.
*   **Turso**: Edge-first database based on libSQL (SQLite fork). Great for low-latency applications.
*   **MongoDB Atlas**: Managed NoSQL database with a permanent free tier (Shared Cluster).
*   **Upstash**: Serverless Redis, Kafka, and Vector databases with a "pay-as-you-go" free tier.
*   **TiDB Cloud**: Fully managed MySQL-compatible serverless database with a strong free offering.

## 4. Useful Public APIs (Data & Utilities)
*   **JSONPlaceholder**: Fake Online REST API for Testing and Prototyping.
*   **OpenWeatherMap**: Free tier for current weather data (requires API key).
*   **NewsAPI**: Search and retrieve live articles from all over the web (free for development).
*   **REST Countries**: Get information about countries via a RESTful API.
*   **NASA APIs**: Access NASA's data including imagery, Mars rover data, and more.
*   **PokeAPI**: All the Pokémon data you'll ever need in one place.

## 5. Design & UI Resources
*   **Unsplash API**: High-quality free images and photos.
*   **Font Awesome**: The most popular icon set, with a solid free version.
*   **Lucide Icons**: Beautiful & consistent icons, 100% open-source.
*   **Excalidraw**: Virtual whiteboard for sketching hand-drawn like diagrams.
